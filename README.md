# Neural Network Based Credit Risk Prediction
This project implements a Multi Layer Perceptron (MLP) neural network to predict credit risk.

## Tools Used
Python
TensorFlow
Pandas
Scikit-learn
Matplotlib

## Dataset
German Credit Dataset

## Steps
1 Data preprocessing
2 Feature encoding
3 Feature scaling
4 MLP architecture
5 Model training
6 Performance evaluation

## Evaluation Metrics
Accuracy
Precision
Recall
F1 Score
Confusion Matrix
