# Neural Network Based Credit Risk Prediction
This project implements a Multi Layer Perceptron (MLP) neural network to predict credit risk.

## Tools Used
Python, TensorFlow, Pandas, Scikit-learn, Matplotlib

## Dataset
German Credit Dataset

## Steps
Data preprocessing, Feature encoding, Feature scaling, MLP architecture, Model training, Performance evaluation

## Evaluation Metrics
Accuracy, Precision, Recall, F1 Score, Confusion Matrix
